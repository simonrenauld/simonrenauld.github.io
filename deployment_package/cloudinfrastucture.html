Cloud Infrastructure & Homelab

Building a personal knowledge base and mastering cloud technologies for independence and control.

About Cloud Infrastructure

Cloud service providers have become essential tools for businesses, offering innovative solutions and powerful data capabilities. However, this reliance can lead to significant costs, especially when retrieving your own data.

As an individual, I'm committed to building a personal Knowledge Base (KB). In our increasingly digital world, having a self-hosted repository provides independence and control over your information.

Why Use Linux?

  • Widespread Use: Powers most web servers and cloud platforms.
  • Open Source: Provides customization opportunities for infrastructure.
  • Command Line Proficiency: Increases adaptability and control.
  • DevOps & Cloud: Essential in cloud workflows and applications.
  • Security: Offers robust tools for securing data infrastructures.

What’s This Project About?

For techies: I'm building a homelab to master cloud infrastructure and enhance my data science knowledge while creating a knowledge base that grants independence from traditional cloud providers.

For non-techies: I’m building a low-cost server because I love to learn and keep control of my data. This setup allows me to run personal applications with the same usability but with added privacy and control.
